# Project Name: To-Do Site

## Overview

The To-Do Site project is a simple web application designed to help users manage their tasks effectively. With a clean and intuitive user interface, users can add, edit, and delete tasks, mark tasks as completed, and organize their to-do lists.

## Features

- **Task Management**: Add, edit, and delete tasks with ease.
- **Mark as Completed**: Keep track of completed tasks.
- **User-Friendly Interface**: Intuitive design for a seamless user experience.

## Technologies Used

- HTML
- CSS
- JavaScript
- NodeJS
- ExpressJS
- MongoDB
